## üîÑ **HOW TO ADD MORE ICONS**

### Process:
1. Visit the library website for your variation
2. Search for the icon you need
3. Copy the SVG code
4. Paste into your HTML
5. Adjust size and color as needed

### Example (Heroicons):
```html
<!-- From Heroicons.com -->
<svg width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
    <path d="..."/>
</svg>

<!-- Customize color -->
<svg width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="#0069BD" stroke-width="2">
    <path d="..."/>
</svg>
```
